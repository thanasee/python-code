# Collection of Python scripts for VASP output and related tasks.

1. Extract and analyze the variables of thermal transport properties in the lattice part from the HDF5 output files of Phono3py
2. Calculate structural properties (e.g., distance) from structure files in VASP format
3. Extract and plot the variables of mechanical properties (e.g., elastic tensor) from output files in VASP format
4. Prepare structure files for various calculations with the VASP calculator
